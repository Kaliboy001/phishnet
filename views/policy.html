PhishNet Usage Policy

1. Purpose:

PhishNet is designed to demonstrate phishing websites and collect educational data, including photos, clipboard information, and location data. It also uses face-recognition technology.

2. Data Collection and Identification:

  • Unique ID: Each bot user is identified by a unique ID.
  • Target Information: Target data is sent to the bot user associated with the unique ID.
  • Change Handling: If the target or ID changes accidentally, the updated information will be sent to the new user or admin.

3. Data Handling:

  • Unassigned ID: If no ID is set, all target information will be directed to the admin bot user.

4. User Responsibilities:

  • Permission: Obtain explicit permission before using the bot on any system and for capturing personal data.
  • Responsible Use: Utilize PhishNet only for educational and research purposes.

5. Disclaimer:

  • Educational Use Only: PhishNet is for educational purposes. The developers are not responsible for any misuse or actions performed using this tool.
  • No Liability: We do not accept responsibility for any outcomes resulting from the use of PhishNet.

6. Technical Issues:

If the bot is unresponsive, verify the status at https://phish.onrender.com/ping.

  • Windows Command: Invoke-WebRequest -Uri https://phish.onrender.com/ping | Select-Object -ExpandProperty content
  • Linux Command: curl -s https://phish.onrender.com/ping
